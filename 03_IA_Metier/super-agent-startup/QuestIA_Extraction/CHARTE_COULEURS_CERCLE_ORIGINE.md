# Charte Graphique : CERCLE ORIGINE

Ce document définit l'identité visuelle pour le futur site immersif QuestIA, basée sur la charte "Cercle Origine".

## 1. Palette de Couleurs (Hex & Sémantique)

| Nom | Hex | Usage Suggéré |
| :--- | :--- | :--- |
| **Noir Onyx Mat** | `#0A0A0A` | Fond principal, sections sombres, texte sur fond clair. |
| **Bordeaux Prestige** | `#630D16` | Accents de luxe, overlays, boutons secondaires, états hover. |
| **Or Liquide** | `#D4AF37` | Éléments interactifs, icônes, bordures fines, titres prestige. |
| **Gris Profond** | `#1C1C1C` | Panneaux, cartes, fonds secondaires, éléments UI. |
| **Phantom White** | `#F2F2F2` | Texte principal, titres sur fond sombre, contrastes. |

## 2. Typographies

- **Titres (H1, H2) :** `Cinzel` (Serif) - Majestueux, classique, luxe.
- **Corps de texte :** `Inter` (Sans-serif) - Lisibilité, modernité, interface.
- **Accents (Badge, HUD) :** `Syne` ou `ui-monospace` - Technique, IA, précision.

## 3. Style Visuel & Effets

- **Gradients :** Utiliser des dégradés du Noir Onyx vers le Bordeaux Prestige pour la profondeur.
- **Effet Or :** Utiliser `linear-gradient(to right, #FFF, #D4AF37, #FFF)` pour les titres importants.
- **Bordures :** Bordures fines de 1px avec l'Or Liquide à faible opacité (0.2 - 0.3).
- **IA Pulse :** Utiliser l'Or Liquide pour les animations de "pulse" des agents IA.
- **Boutons :** Style "Luxe" avec bordure or, texte blanc, effet hover plein or avec ombre portée.

---

## 4. Application au Projet QuestIA

### HUD & Interface Simulation
- Remplacer le Cyan (`#00e5ff`) par l'**Or Liquide** (`#D4AF37`).
- Remplacer le Violet (`#7b5cff`) par le **Bordeaux Prestige** (`#630D16`).
- Le fond restera le **Noir Onyx Mat**.

### Portraits & Villes
- Utiliser des "Tints" basés sur le Bordeaux et l'Or pour harmoniser les photos Unsplash avec la charte.

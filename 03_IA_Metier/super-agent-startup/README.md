## 🛠️ Technologie & Méthodologie

### 🧠 Engine Comportemental
Notre moteur ne se contente pas d'analyser vos mots, il décode l'invisible :
*   **Profilage DISC & PCM** : Détection en temps réel du profil de l'interlocuteur et score d'adaptabilité.
*   **Stress Bio-Feedback** : Analyse du spectre vocal, des silences et de l'assurance pour détecter les micro-signaux de faiblesse.
*   **Fine-tuned LLM** : Modèle d'IA propriétaire entraîné sur +10 000 heures de négociations complexes.

### 📊 Leadership Dashboard
Un outil de pilotage pour les managers et DRH :
*   **Alerte Détresse** : Notification proactive avant le décrochage d'un collaborateur.
*   **Scoring RH Prédictif** : Preuves objectives de l'agilité comportementale pour le recrutement.
*   **Connectivité CRM** : Intégration native avec **HubSpot** et **Salesforce** pour des répétitions basées sur des deals réels.

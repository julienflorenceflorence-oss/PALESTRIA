# Data Analysis Framework

This framework defines the methodology for analyzing business data (Ads, SEO, Sales, etc.).

## 📊 Subjects of Analysis
- **Advertising**: Google/Meta Ads performance.
- **Prospecting**: Phoning, LinkedIn outreach, cold mailing.
- **SEO/Organic**: Search rankings and traffic quality.
- **Sales & CRM**: Conversion rates, sales cycle, churn.
- **Feedback**: Form results, newsletter engagement.

## 🔍 Diagnostic Questions (MANDATORY)
Before starting any analysis, the agent MUST ask the user:
1. **Target Audience**: Who is the ideal customer?
2. **Analysis Objective**: What specific goal are we trying to reach?
3. **Context**: Any commercial event, restructuring, or macro-economic change?
4. **Timeline**: What is the date range for the analysis?
5. **Data Sources**: Which specific files or APIs are available?
6. **KPIs**: Are there specific metrics you want to calculate (ROI, CAC, ROAS)?

## 📤 Expected Outputs
- **Executive Summary**: Factual and surgical.
- **Detailed Data Report**: Graphs, tables, and cited sources.
- **Action Plan**: "Who does what?" with difficulty levels for each task.
- **KPI Barometer**: Evaluation of current state vs. goals.

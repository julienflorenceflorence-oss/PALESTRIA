# Presentation Freelance - Julien FLORENCE

Ce projet contient la landing page de présentation des services freelance de Julien FLORENCE, Architecte de Systèmes Intelligents & Coach de Performance.

## 🚀 Aperçu
- **Esthétique :** Édition Prestige (Noir Profond & Or Primaire).
- **Architecture :** Multi-pages (Accueil, Méthode, Services, Contact).
- **Technologies :** HTML5, CSS3 (Glassmorphism, Animations tactiques), JavaScript.

## 📂 Structure du projet
- `index.html` : Accueil immersif avec citation de Denis Waitley.
- `methode.html` : Détail de la méthodologie (DISC, PCM, Vision Handball).
- `services.html` : Scénographie interactive "Command Center" pour les offres.
- `contact.html` : Carré d'actions premium pour la prise de contact.
- `prestige.css` : Charte graphique unifiée.

## 🛠️ Déploiement
Ce site est conçu pour être hébergé sur **GitHub Pages**.

1. Créez un dépôt nommé `presentation-freelance` sur votre compte GitHub.
2. Poussez les fichiers de ce dossier vers le dépôt.
3. Activez GitHub Pages dans les réglages du dépôt (Branche `main`, dossier `/root`).

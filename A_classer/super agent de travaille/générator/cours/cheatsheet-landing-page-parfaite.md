## La Checklist (Indispensable)

- [ ] Proposition de valeur copywritée
- [ ] Plusieurs éléments de réassurance
- [ ] La page a un unique objectif (un CTA)
- [ ] LP orientée bénéfices et non features
- [ ] Mobile-first (80% du trafic est mobile)
- [ ] Les images ont été compressées
- [ ] La page fait apparaître de l'humain
- [ ] Mentions légales & Politique de confidentialité
- [ ] Bannière de cookies (Axeptio)
- [ ] Navbar & footer simplifiés
- [ ] Les balises de suivi (Meta, Google, etc.)

---

## Template du Héro

Structure recommandée pour la section héro :

1. **Logo / Nom de marque** (ex : AdsBack)
2. **CTA Principal** (bouton visible en haut)
3. **Proposition de valeur** (titre principal)
4. **Sous-texte descriptif**
5. **Liste d'avantages** :
   - Avantage n°1
   - Avantage n°2
   - Avantage n°3
6. **Bouton CTA Principal**
7. **Réassurance** (ex : avis étoilés, note type Trustpilot)
8. **Logos de partenaires / clients** (ex : Station F, Adtech, etc.)
9. **Visuel / Image** à droite

## Le Call to Action

- [ ] Formule : **Verbe d'action + valeur**
- [ ] Utiliser une flèche dans le CTA
- [ ] Contrer les objections sous le CTA
- [ ] Utiliser une couleur vive qui contraste

**Exemple :**

> → **Start building your landing**
>
> Free trial - No Credit Card

# MÉTHODOLOGIE : CALCUL ET MAXIMISATION DU ROI

Utilisez ce guide pour justifier la rentabilité de vos actions dans votre Note d'Opportunité.

### 1. LES 3 LEVIERS POUR BOOSTER LE ROI
Pour augmenter le ROI d'une entreprise, vous ne pouvez jouer que sur 3 leviers :
1.  **Baisser les coûts d'acquisition (CAC) :** Mieux cibler les pubs, faire du SEO.
2.  **Augmenter le taux de conversion (CR) :** Améliorer le site web (UX), rassurer le client.
3.  **Augmenter la valeur vie client (LTV) :** Faire racheter les clients actuels (Emailing/Fidélité).

### 2. TABLEAU DE CALCUL TYPE (À REPRODUIRE)
Voici comment présenter votre rentabilité demain :

| Élément | Valeur Estimée | Source / Justification |
| :--- | :--- | :--- |
| **Investissement Total** | 3 000 € | Budget Ads (2k) + Setup Freelance (1k) |
| **Trafic Estimé** | 2 000 clics | Basé sur un CPC moyen de 1,00 € |
| **Taux de Conversion** | 3 % | Moyenne du secteur (Prudent) |
| **Nombre de Ventes** | 60 ventes | 2 000 clics x 3% |
| **Panier Moyen** | 150 € | Historique de l'entreprise |
| **CA Généré** | 9 000 € | 60 ventes x 150 € |
| **ROI (Projet)** | **200 %** | (9 000 - 3 000) / 3 000 |

### 3. VOCABULAIRE FINANCIER (Points bonus "Style")
*   **Seuil de rentabilité :** Le moment où le CA couvre exactement les dépenses.
*   **Coût d'opportunité :** Ce que l'entreprise perd en ne faisant PAS l'action (ex: "Chaque mois sans SEO nous coûte 500 prospects gratuits").
*   **Scalabilité :** Capacité à doubler le budget tout en gardant la même rentabilité.

---
**CONSEIL DU COACH :** En conclusion de votre note, ne dites pas "Je pense que ça va marcher". Dites "Au vu du modèle de rentabilité exposé, l'action présente un risque maîtrisé pour un ROI cible de X%".

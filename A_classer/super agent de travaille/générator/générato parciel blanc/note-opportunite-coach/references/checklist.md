# Checklist de Réussite Avant Rendu

Avant d'exporter en PDF, vérifiez CHAQUE point :

- [ ] **Nom du fichier** : `Date-Promo-Nom-Prénom.pdf` (ex: 20260316-M2-Dupont-Jean.pdf)
- [ ] **Longueur** : Le document fait maximum 3 pages (hors titre).
- [ ] **Format** : Police Arial 12.
- [ ] **Problématique** : J'ai une phrase CLAIRE avec des chiffres et une date (SMART).
- [ ] **Recommandations** : J'en ai au moins 2 ou 3 majeures.
- [ ] **Tableau d'Actions** : Chaque action a un Budget estimé.
- [ ] **Tableau d'Actions** : Chaque action a une Deadline (Quand ?).
- [ ] **Tableau d'Actions** : Chaque action a un KPI précis.
- [ ] **Orthographe** : J'ai relu deux fois pour l'orthographe (1 point gratuit !).
- [ ] **Ton** : Est-ce que je parle à un CEO ou à un prof ? (Réponse : CEO ! Soyez "Business Oriented").

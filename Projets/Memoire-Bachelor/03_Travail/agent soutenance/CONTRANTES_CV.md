# Charte de Réalisation - CV Julien FLORENCE (Édition Prestige)

Ce document définit les contraintes strictes pour toute modification du CV.

## 1. Design & Esthétique (Édition Prestige)
*   **Code Couleur** : Fond noir (#000 / #0F1115), texte lumière (#f4f4f4), accents Or Prestige (#D4AF37).
*   **Arrondis (Bordures)** : **STRICTEMENT 10px** pour tous les éléments (Boutons CTA, Cadres d'expériences, Conteneur photo, Cartes de diplômes).
*   **Typographie** : 'Cinzel' pour les titres et boutons, 'Montserrat' pour le corps de texte.
*   **Boutons (CTA)** : Style arrondi 10px, bordure Or 2px, effet survol (hover) avec remplissage Or.

## 2. Intégrité du Contenu
*   **Stabilité du Texte** : Une fois validé, aucun texte (expériences, missions, titres) ne doit être modifié ou simplifié sans demande explicite.
*   **Profil Psychologique** : Doit toujours inclure les traits détaillés (Rationnel, Organisé, Extraverti) et la section stratégique (Posture girafe, etc.).
*   **Dates** : Format standardisé **MM/YYYY – MM/YYYY** (ex: 09/2000 – 02/2001).

## 3. Responsive & Affichage
*   **Verrouillage Portrait** : Pour tous les écrans inférieurs à 13 pouces (Tablettes et Mobiles), la lecture horizontale est interdite via un overlay de blocage "Prestige".
*   **Photo de Profil** :
    *   **Taille Desktop** : Largeur fixe (100% du conteneur aside de 35%).
    *   **Taille Mobile** : 50% de la largeur de l'écran, centrée.
    *   **Style** : Toujours encadrée par une bordure Or de 2px avec un arrondi de 10px.
*   **Boutons bas de page (CV)** : Non-fixes sur la page CV, se placent après le dernier contenu.
*   **Boutons bas de page (Diplômes)** : Dock flottant premium sur mobile.

## 4. Navigation spécifique
*   **Mobile** : Le bouton "Portfolio Diplômes" est supprimé du bas de page et remplacé par un bouton unique "Portefolio" placé sous la section "Certification Conseiller Immobilier (2018)".

*   **Design :** Intégration de la police *Bricolage Grotesque*, dégradés QuestIA et logo haute résolution.

## 🚀 Installation rapide
1.  Ouvrez la [Preview de la Signature](https://julienflorenceflorence-oss.github.io/questia-signature/) (si déployée sur GitHub Pages).
2.  Cliquez sur le bouton **"Copier pour Gmail"**.
3.  Allez dans vos **Paramètres Gmail** > Section "Signature".
4.  Collez (Ctrl+V) et enregistrez.

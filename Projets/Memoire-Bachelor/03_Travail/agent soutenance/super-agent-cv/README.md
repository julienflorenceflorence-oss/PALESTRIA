# Super-Agent : Création Prestige (CV & Signature)

Cet agent gère la production automatisée de vos documents d'identité professionnelle.

## 🛠️ Skills Disponibles
1.  **cv-template-pro** : Création de CV Web et Dossiers de Compétences (Design 10px, Portrait-Only, svh).
2.  **signature-pro** : Création de Signatures Mail et Templates HTML (Tableaux, 600px, Inline CSS, Placeholders).

## 🚀 Activation
Pour utiliser une fonction, tapez :
`activate_skill cv-template-pro` ou `activate_skill signature-pro`

## 📋 Rappel des Standards
- **Arrondis** : 10px pour le Web.
- **Email** : Toujours en Inline CSS pour éviter le blocage des boîtes mail.
- **Images** : Toujours via URL Cloud (Zéro pièce jointe).

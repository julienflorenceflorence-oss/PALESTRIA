# Skill : Premium GIF Production (High-Fidelity)

Ce skill définit les règles pour transformer une vidéo ou une animation en un GIF de qualité "Prestige".

## 1. Spécifications Techniques (La Base)
- **Fréquence d'images** : Entre 24 et 30 fps (Fluidité absolue).
- **Résolution** : Minimum 720p (HD). Pas de flou de pixel.
- **Boucle (Looping)** : Seamless Loop obligatoire (Transition invisible).
- **Poids** : < 5 Mo pour le Web / < 15 Mo pour les Réseaux Sociaux.

## 2. Qualité Visuelle & Colorimétrie
- **Dithering** : Algorithme Floyd-Steinberg pour des dégradés lisses.
- **Palette** : Palette adaptative par image (256 couleurs optimisées).
- **Netteté** : Léger filtre "Sharpen" post-compression.

## 3. Direction Artistique
- **Look & Feel** : Micro-histoire élégante, symétrie parfaite.
- **Cinémagraphe** : Mouvement partiel pour un effet luxe.

## 4. Workflow de Production
1. **Montage** : Premiere Pro (Précision frame).
2. **Animation** : After Effects (Easy Ease).
3. **Export** : Photoshop (Save for Web - 256 col).
4. **Optimisation** : Gifsicle (Réduction poids).

# LÉGENDE DES COULEURS
- <span style="color:red">**Rouge Gras**</span> : Points importants ou de vigilance (Risques, Menaces, Faiblesses).
- <span style="color:green">**Vert Gras**</span> : Points positifs (Forces, Opportunités, Gains, Succès).
- <span style="color:yellow">**Jaune Gras**</span> : Points d'influence (Marché, Évolutions, Pivots Stratégiques).
- <span style="color:blue">**Bleu Gras**</span> : Points d'observation (Faits, Données neutres, Statistiques).

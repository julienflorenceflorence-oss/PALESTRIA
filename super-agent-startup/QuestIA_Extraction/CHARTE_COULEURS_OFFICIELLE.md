# Charte Couleurs Officielle - QuestIA Immersif

Ces couleurs sont les seules autorisées pour la construction du nouveau site QuestIA.

## 1. Couleurs de Base (Interface)

| Variable | Hex | Description |
| :--- | :--- | :--- |
| **Fond Principal** (`--bg`) | `#05070d` | Noir bleuté profond |
| **Fond Secondaire** (`--bg-2`) | `#0a0e18` | Bleu nuit |
| **Texte Principal** (`--text`) | `#eef1f7` | Blanc bleuté froid |
| **Texte Muted** (`--muted`) | `#8a93a8` | Gris acier |

## 2. Couleurs d'Accentuation & Statuts

Utilisées pour les indicateurs de performance, les statuts et les éléments interactifs :

| Variable | Hex | Usage |
| :--- | :--- | :--- |
| **Accent Principal** (`--accent`) | `#00e5ff` | **Cyan électrique** : Actions primaires, succès, IA. |
| **Corail** (`--coral`) | `#ff5a6a` | Erreurs, risques élevés, pression. |
| **Ambre** (`--amber`) | `#ffb547` | Avertissements, minuteurs. |
| **Lime** (`--lime`) | `#b8ff3d` | Validation des niveaux, scores positifs. |
| **Violet** (`--purple`) | `#7b5cff` | Mode groupe, badges spéciaux. |

---

## Directives d'Application
- **Strict Adherence :** Aucune autre couleur ne doit être introduite.
- **Contraste :** Respecter les ratios de contraste définis dans le skill `ux-responsive-expert`.
- **Hiérarchie :** Utiliser le Cyan (`#00e5ff`) comme guide visuel principal pour l'interactivité.

1. Architecture de l'Agent (Modular System)

A. Orchestration LLM

  • Framework : LangGraph (pour les cycles décisionnels complexes).
  • Model : GPT-4o (Reasoning) + Mistral Large (Privacy-focus).
  • Memory : Redis pour le stockage des sessions de chat temps-réel.

B. Vector Engine (RAG)

  • Database : Pinecone ou Weaviate.
  • Embeddings : text-embedding-3-small.
  • Content : Livrets d'accueil (MD/PDF), Inventaire Vins, Règlementations locales.

2. Module Sniper (Data Recovery)

C'est le module de conversion "Anti-OTA". Il doit identifier les patterns d'abandon.

  • Tracking : Injection d'un script JS origin-tracker.js sur le site membre.
  • Trigger : Analyse du Exit Intent et du Time on Page.
  • Logic : Si user_score > 0.8 (visiteur qualifié) ET intent == 'exit', déclencher le popup d'offre directe via WebSocket.
